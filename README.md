# Скрипт автоматического включения школьных звонков на Python

## Запуск

```bash
screen -dmS bells python3 /root/Bells/bells.py
```

## Для OrangiPi Zero с GPIO

На вахте в прозрачном ящике

* Звонки по времени
* Предварительные звонки
